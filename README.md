# Tic Tac Toe Web Application

This is a responsive Tic Tac Toe game developed using HTML, CSS and JavaScript.

## Features

- Responsive Design
- Two Player Game
- Winner Detection
- Draw Detection
- Restart Game Button
- Simple and Modern UI

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project
2. Open index.html file in browser
